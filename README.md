# LingoLoop

LingoLoop is a lightweight web application for listening practice with text-to-speech playback.

Users can paste any text, choose a narrator language and voice, adjust reading speed, set repeat count, and switch between full-text playback and sentence-by-sentence mode.

## Features

- Paste and read any text with browser-based text-to-speech
- Select narrator language and available voice
- Adjust reading speed
- Configure repeat count
- Set pause between repeats
- Sentence-by-sentence playback mode
- Light and dark theme support
- Local settings persistence via `localStorage`

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4
- Web Speech API
